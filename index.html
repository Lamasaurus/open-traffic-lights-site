What is it?

Open traffic lights is a Proof-of-Concept for publishing traffic lights data as Open Data. This is done using Linked Data technologies and real-time publication of data. Also historical data is accessable for end-users. This page shows how to use the data with a demonstrator. If you want to read more how to publish real-time data, check out the specification page. This project is part of the City of Things programme .
